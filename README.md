# Notes dashboard

A very simple markdown post-it dashboard.

Made with React + Webpack + Koa + Knex + Objection ROM

## Installing

```bash
git clone https://github.com/cristianchelu/notes-dashboard.git
cd notes-dashboard
npm install
```

## Running
```bash
# Production
npm run build
NODE_ENV=production node src/server/index.js

# Testing
npm run dev:db
npm run dev
```
